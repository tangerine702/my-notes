安装完了我们也得用用，接下来介绍命令

首先还是打开数据库

你可以用以下两种方式打开：

  1. 右键单击我的电脑，点击属性，选择高级系统设置，点击右下角的环境变量，在系统变量那一栏找到Path这个选项，双击进入看看有没有关于MySQL8.0\bin 这种路径，如果有，那就不用管了，如果没有，找到MySQL命令行客户端（MySQL Command Line Client）右键，点击属性，有一个打开文件位置，复制路径进入Path粘贴即可
  2. 你可以直接点击MySQL命令行客户端（MySQL Command Line Client）进入（滑稽）

接下来就是真正的命令了

# 基础命令介绍

点击打开MySQL命令行客户端之后，会出现一个黑色的窗口，上面闪动着一个提示：

Enter password：

此时点击下窗口，键盘输入你安装数据库时设置的密码，

如果正确，它会显示：Welcome to the MySQL巴拉巴拉的东西，

然后看到最后一行有一个mysql>

好了，他在提示你输入指令(￣y▽,￣)╭

首先我们先输入这一段指令来查看你现在有什么数据库：

show databases；

他会出现一个这种表格：

Database
information_schema
mysql
.......
sys

以上都是你安装数据库后系统自动创建的，

那我也创建一个自己的数据库呗😏

输入指令：

Create Database 数据库名称;

例如我输入：Create Database demo;

回车他就会提示你“Query OK，1 row affected（0.00sec ）”

看到这个提示就说明你名为demo的数据库创建好了

这个时候我们再次输入命令：show databases;查看所有的数据库

就会发现你刚创建的数据库demo了

创建完数据库了，不得存点东西？🤔

来了

想往哪里存数据，首先得选择

输入use 数据库名称

选择数据库

例如我选择demo这个数据库，输入use demo

系统提示：Database changed

这就代表选择成功

接下来在数据库里存放一张数据表

这个命令有点长：

create table 数据表名称
	         (
				字段名称1 数据类型，
				字段名称2 数据类型，
				........
	          );
1
2
3
4
5
6

还是那我demo数据库举例子

我在里面存一张teacher教师表，

字段1我设置id ，数据类型设置int类型，

字段2我设置name，数据类型设置varchar，

这里的int和varchar都属于数据类型，分别存储数字类型和字符类型

以下为实操代码：

create table teacher
		(
			id  int(5),
			name varchar(10)
		);
1
2
3
4
5

依然显示“Query OK，1 row affected（0.00sec ）”时表示创建成功

特别注意：use 数据库名称和create table 新建的数据表名称两步是需要连起来用的，不然你直接使用create table 新建的数据表名称是没用的，系统不知道在哪个数据库中新建数据表，所以会报出“ERROR”错误

那我们创建了数据表，它得给我看看它里面是什么样的吧（给我康康！~）

输入指令：desc 数据表名称

它会显示一个表和一行提示，类似这种：

Field type null Key Default Extra
id int YES NULL
name varchar（10） YES NULL

2 rows in set （0.02 set）

.....

既然表也创建好了，我往里添加一些记录，比如老师我聘请了张三，那怎么加入数据表呢？

首先我们要查看数据表里有什么数据，输入命令

select *	
from 要查询的数据表名称;
1
2

注意上面命令分为两行输入，第一行代码输完以后理论上没有分号作为结尾他是会换行继续让你输入的，这样更美观也更容易发现错误

但是输入完发现没有一条记录（😂）

这时我们开始插入数据

老师我聘请了张三，所以要给张三一个独属于他自己的id号，叫01吧

那我们插入张三的数据：

输入命令：

insert into teacher(id,name)
values('01','张三');
1
2

依旧显示：Query OK, 1 row affected (0.01 sec)这一条提示

表明插入成功

那怎么查看我里面存着的张三的数据呢？

select *
from teacher;
1
2

就显示出来了这样一个表格：

id name
01 张三
