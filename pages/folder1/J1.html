它具有简单性、面向对象、分布式、健壮性、安全性，平台独立性和强大的跨平台可移植性，应用广泛

# 布置运行环境：

好的让我们来布置环境运行这段代码吧！

首先下载一个JDK，点击此处跳转到下载界面 (opens new window)

点击下载最下面那个Windows x64 Installer 159.71 MB后面的下载得到一个安装文件(.exe)

然后什么都不用改，直接无脑下一步即可

提示下载完后找到安装位置(一般为C盘的Program File\Java\jdk-xx.x.x)

复制这个目录，鼠标右键此电脑，选择属性→高级系统设置→环境变量，找到系统变量那一栏(用户变量下面那一栏)新建变量，第一行命名为JAVA_HOME,第二行粘贴刚才复制的目录，点击确定

按win键+R键(win键一般在Alt左边那个windos图标的按键)输入cmd

写入命令java、javac、java -version都有反应而不是出现'javac' 不是内部或外部命令，也不是可运行的程序或批处理文件这种提示即可

# 软件安装

接下来是软件

这里使用对小白十分友好的IDEA，点击此处即可直达官网 (opens new window)

这里专业版对入门的我们基本用不到，所以我们下载免费的社区版（有Community标注的版本）点击下载

下载完成后我们双击安装包进行安装

看到欢迎界面，点击next下一步

设置安装路径，建议安装到C盘以外的路径去（不要使用有中文的路径），比如我这里选择D盘根目录，下一步

勾选Create Desktop Shortcut创建桌面图标，勾选Create Associations中的所有后缀名，点击下一步

点击install安装即可

软件运行后弹出允许许可，勾选后点击Continue继续

点击Don‘t send，等一会后即进入IDEA启动界面

# 安装中文插件

进来后能看到是英文版，以下步骤可设置为中文

  1. 点击Plugins，在搜索框里搜索Chinese，找到中文语言包（一般在第二个），点击绿色的install安装
  2. 安装完成后提示重启IDEA完成设置，点击Restart IDE，弹出窗口选择Restart重启即可

# 创建项目

重启完后即可看到创建项目按钮，点击新建项目，看下上方的项目SDK有没有搜索到（一般会自动检索你有没有安装）如果没有的话找到安装目录选择一下即可，点击下一步继续

取消从模板创建的选项，下一步

填写项目名称和项目存储地址，下一步等待IDEA构建项目成功后即可进入软件界面

# 软件小设置

首先找到关闭按钮下面有一个齿轮按钮，点击它，下拉框内出现项目结构，点击进入项目结构，看到项目SDK已经显示出你的版本号了

当然你要是没看到的话可以点击下面的SDK，找到左上角的加号进行添加

# 编写第一个HelloWorld程序

在完成上面两步后，我们找到IDEA左侧的项目列表，找到Src文件夹，鼠标右键找到新建→Java类，输入类名即可创建一个空的类，在空的类的大括号内输入以下代码：

public static void main(String[] args) {
		System.out.println("Hello World!");
	}
